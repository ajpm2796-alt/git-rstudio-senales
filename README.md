# git-rstudio-senales
# Alessandro

Repositorio de práctica para aprender Git, GitHub y RStudio.

## Objetivo

Este repositorio contiene una práctica básica para aprender a crear un proyecto en RStudio, guardar cambios con Git y subirlos a GitHub.

## Primer ejemplo

Se generó una señal senoidal de 5 Hz usando R.

## Archivos principales

- `scripts/01_senal_basica.R`: script que genera una señal senoidal.
- `figuras/senal_basica.png`: imagen generada por el script.
- `Alessandro.Rproj`: archivo del proyecto de RStudio.
